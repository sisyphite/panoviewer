# 全景图查看器（Panorama Viewer）使用说明
这是一个简单实用的本地全景图查看工具。它不需要联网，也无需安装任何软件或插件。您只需在电脑上双击打开这个网页文件，就能在浏览器中直接浏览全景照片，完全离线且安全隐私。

## 一、 主要功能与特点
1. **免安装、纯本地运行**：无需搭建服务器，双击 HTML 文件即可在浏览器中运行，且断网环境下同样可用。
2. **轻量与兼容性**：不加载外部资源，启动迅速；支持绝大多数支持 WebGL 的现代浏览器。
3. **批量载入与导航**：支持一次性导入一整组全景图片，并提供步长控制，方便快速筛选或对比连续拍摄的全景图。
4. **内置坐标测试图**：提供带有经纬网格和方向标记（东、南、西、北、天顶、天底）的虚拟全景图，便于功能测试与视角对齐。


---

## 二、 快速上手步骤

### 1. 运行工具
在您的电脑上双击`viewer.html`文件，或将其拖入任意现代浏览器（如 Chrome、Edge、Firefox、Safari 等）中打开。

### 2. 视角交互操作
打开网页后，您可以通过以下方式交互浏览全景图：
* **旋转视角**：在画面上**按住鼠标左键并拖拽**，即可进行 360° 环视和上下俯仰观察。
* **画面缩放**：通过**滚动鼠标滚轮**，可以放大（拉近）或缩小（拉远）视野（FOV 调整范围为 30° 至 100°）。
* **状态查看**：网页左下角会实时显示当前的参数指标：
  * `yaw`：偏航角（水平旋转角度）
  * `pitch`：俯仰角（垂直倾斜角度）
  * `fov`：视场角（视野范围大小）

### 3. 载入您自己的全景图片
* **导入文件夹**：点击右上角的 **“选择图片文件夹”** 按钮，在弹出的系统窗口中选择存放全景图片的文件夹，并授予浏览器读取权限。
* **自动筛选与排序**：工具会自动过滤出文件夹中支持的图片格式（`.jpg`、`.jpeg`、`.png`、`.webp`、`.bmp`、`.gif`），并按文件名进行自然升序排序。
* **格式建议**：本工具设计用于查看**等距柱状投影（Equirectangular / 通常宽高比为 2:1）**格式的标准全景图。普通视角的非全景图片在载入后会出现拉伸变形。

### 4. 切换图片与步长控制
导入文件夹后，底部的导航栏会显示当前正在查看的文件名和图片索引（例如 `1 / 15`）：
* **切换图片**：
  * **鼠标操作**：点击底部导航栏的 **◀**（上一张）或 **▶**（下一张）按钮。
  * **键盘操作**：按键盘上的 **左方向键 (←)** 或 **右方向键 (→)** 快速切换。
* **步长设置**：
  * 在底部的“步长”输入框中，您可以修改跳转间隔（默认值为 2）。
  * 比如：若步长设为 `2`，按“下一张”时当前索引会直接 `+2`（如从第 1 张直接跳到第 3 张）。如果临近图片末尾，跳转会自动限制在最后一张，避免卡住。

### 5. 恢复测试图
如果您想清除当前加载的本地文件夹，或想校准方向，可以点击右上角的 **“恢复测试图”**。工具会重新生成一张包含水平角度和四个基准方向（东 E、南 S、西 W、北 N）的虚拟 3D 全景网格图。

---

## 三、 注意事项
* **本地隐私安全**：通过本工具选择本地文件夹时，图片数据仅在您的浏览器本地内存中处理、渲染并生成临时的 Blob 链接，**没有任何数据会被上传至外部网络**，您可以放心处理私密照片。
* **键盘焦点限制**：当您的鼠标光标正在“步长”输入框中编辑数字时，键盘的左右方向键会用于移动文字光标，此时不会触发全景图切换。编辑完成后，鼠标点击一下全景图背景即可恢复方向键切换功能。
